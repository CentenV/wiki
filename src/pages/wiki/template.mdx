# Templates for every page type

## Content Pages
Metadata defining each page:

```
---
title: "Page Title/Topic"

    Title of content page. What the article/page is about


layout: "@layout/Content.astro"

icon?: "name_of_icon_file.svg"

    Specifies what icon svg to use located in the src/icons folder


banner_icon?: "path_to_icon.svg"

banner_image?: i

---
```
