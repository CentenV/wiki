# Templates for every page type

Every page has some boilerplate that is unavoidable and is important to maintain consistency amongst the wiki pages.

## Content Pages

### Metadata defining each page:

```
---
title: string
layout: "@layout/Content.astro"
icon: string?
icon_invert: bool?
---
```

**title**: Title of content page. What the article/page is about \
**layout**: Constant. Defines what layout to use \
**icon**: Specifies what icon svg to use located in the src/icons folder \
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Ex: "name_of_icon_file.svg" \
**icon_invert**: Invert the specified icon. \
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Default: true
